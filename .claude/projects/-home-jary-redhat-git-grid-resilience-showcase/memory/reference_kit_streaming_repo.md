Reference repo for Kit App Streaming: `/home/jary/redhat/git/nvidia-kit-app-streaming`

- Working example of streaming IsaacSim to browser from OpenShift via WebRTC
- Contains IsaacLab references that are NOT needed for this project
- Key difference for grid-resilience: stream just the scene view instead of the full app window (there's a parameter to control this)
- Use this as the basis for the Kit substation streaming setup
